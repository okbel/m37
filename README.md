## Usage

```bash
# Installation with `gatsby-cli`
gatsby new my-site https://github.com/itstimetogoleft/m37-gatsby-starter

# Installation with `git clone`
git clone my-site https://github.com/itstimetogoleft/m37-gatsby-starter.git
cd my-site
yarn

# To develop
yarn develop

# To build
yarn build

```
